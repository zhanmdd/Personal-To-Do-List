## Overview
A To-Do list Web App

## Main Features
- Custom Categories: Creating, Deleting, Updating and Storing
- Tasks (inside the category): Creating, Deleting, Updating and Storing

## Technologies
- HTML5
- CSS3
- Javascript: jQuery
- Node.js: express, body-parser, EJS, lodash
- MongoDB: Mongoose
- Heroku
