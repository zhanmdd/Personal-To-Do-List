# Personal-To-Do-List with MongoDB and and Node.JS

A modern to-do-list web application with the essential functinoalities:
- Creating, Deleting, Updating and Storing lists
- Creating, Deleting, Updating and Storing tasks inside the lists

## The Web App Was Built Using
- HTML5
- CSS3
- Javascript: jQuery
- Node.js: express, body-parser, EJS, lodash
- MongoDB: Mongoose
- Heroku

## Deployment
Website Link: https://todolist-05.herokuapp.com/

(Hosted with Heroku)
